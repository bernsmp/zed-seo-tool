# Zed SEO Cowork Project Folder

Use this folder as the local folder for Carlos's dedicated Zed SEO Cowork Project.

The plugin creates these folders on first setup:

- `clients/` for durable client profiles and URL inventories;
- `imports/` for SEMrush and uploaded CSV files;
- `jobs/` for immutable job inputs and checkpointed batches;
- `exports/` for operator-ready CSV files;
- `incidents/` for sanitized diagnostic reports.

Client and keyword data stays here and is never bundled into the plugin.
